# 百度、高德，WGS84坐标批量转换工具

## 简介
本仓库提供了一个用于百度、高德坐标批量转换的工具。该工具能够帮助用户快速将百度、高德坐标转换为WGS84坐标，适用于需要进行地理坐标转换的各类应用场景。

## 功能特点
- **批量转换**：支持一次性转换多个坐标点，提高工作效率。

- **简单易用**：操作简便，用户只需按照提示进行操作即可完成坐标转换。
